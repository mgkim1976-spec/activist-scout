# 보고서 예시

이 폴더에는 `activist-scout` 로 만든 *공개 가능한* 보고서 예시들이 들어 있습니다.

## 폴더 구조

```
examples/
├── README.md              # 이 파일
└── sample_reports/        # 펀드 매니저급 한 종목 분석 보고서 예시
    ├── deep_dive_021820.md   # 세원정공 (자동차 부품 + 거버넌스 사고 케이스)
    ├── deep_dive_016740.md   # 두올 (행동주의 펀드 14.65% 진입 케이스)
    └── ...
```

## 이 예시들의 목적

1. **결과물 미리 보기** — 직접 실행하기 전에 어떤 형식의 보고서가 나오는지 확인
2. **실제 사례 학습** — 한국 유가증권시장 종목들의 분석 패턴
3. **자동화의 한계 인지** — 보고서 마지막의 *"⚠️ 사람 검증 필수 항목"* 섹션이
   어떻게 작성되는지 확인

## 직접 만들기

```bash
# 예시와 같은 보고서를 직접 생성
python -m activist_scout.deep_dive 021820 \
    --output reports/sewon.md
```
