## [0.1.0] — 2026-05-09

### 첫 공개 버전

이 버전부터 누구나 자유롭게 가져가 사용할 수 있도록 공개됩니다 (MIT 라이선스).

#### 핵심 기능

- 한국 유가증권시장 행동주의 후보 자동 발굴 (8단계 흐름)
- 한 종목 펀드 매니저급 분석 보고서 자동 작성 (13개 섹션)
- 매일 새로운 5% 이상 지분 신고 알림
- 분기별 과거 검증 (시장 평균 대비 수익률)

#### 주요 분류 및 규칙

- 3축 점수 (타깃 매력도 / 매집 흔적 / 법적 지렛대)
- 9가지 등급 분류 (강한 후보 / 유망 / 시장 회의적 / 관심 / 시장 미온적 / 이미 반영됨
  / 신호 신선 / 관망 / 피해야 함)
- "시장 회의적" 하위 분류 — 신고 후 시장 평균 대비 −5% 이하 (역설적 매수 기회)
- "피해야 함" 면제 규칙 — 최대주주 60~65% 구간 + 자회사 합산 가치 또는 자사주
  정책이 우호적인 경우
- 모회사 종속 자회사 자동 식별 (일감 비중 + 모회사 지분율)
- 사외이사 임기 만료 임박 → 변화 가능 시점 추적
- 자회사 합산 가치 계산 (지주회사 보통 30~60% 저평가)
- 인공지능으로 사업보고서 읽어 일감 몰아주기 비중 정량화

#### 단일 종목 보고서 12개 보조 분석

- 자기자본 수익률, 자유 현금 흐름 수익률
- 동종업계 비교
- 외국인 지분 추세
- 임원 보수와 회사 성과 일치도
- 자사주 매입·소각 5년 이력
- 5년 주가 흐름
- 이벤트 발생 확률 × 영향도 매트릭스
