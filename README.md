# activist-scout

**한국 주식 중 "주주 권리를 적극 주장하는 펀드(행동주의 펀드)" 가 관심 가질 만한
종목을 자동으로 찾아주고, 한 종목씩 깊이 분석한 보고서를 30초 만에 만들어 주는
도구.**

> 💡 **이 프로젝트가 어떻게 만들어졌는지** → [STORY.md](STORY.md)
>
> 한 줄짜리 부탁에서 시작해, AI 코딩 도우미([Claude Code](https://claude.ai/code))
> 와 9일간 대화하며 만들어진 분석 도구의 진화 이야기.

---

## 1. 이게 뭔가요?

한국 거래소에 상장된 약 2,500개 종목을 자동으로 훑어서, **"주주 가치 개선을 요구할
만한 외부 펀드(행동주의 펀드)가 캠페인을 걸기 좋은 회사"** 를 추려 줍니다.

추려낸 종목 중 관심 있는 한 종목을 골라 명령어 한 줄을 실행하면, **펀드 매니저가
한 시간 동안 작성할 분석 보고서를 약 30초 만에 자동으로** 만들어 줍니다.

### 행동주의 펀드란?

지분을 사들인 뒤 **"이 회사 자산이 너무 많이 묵혀 있으니 자사주를 사서 소각해라",
"자회사를 분리 상장해라"** 같은 식으로 경영진에게 변화를 요구하는 펀드입니다.
한국에서는 차파트너스, 얼라인파트너스, KCGI 등이 대표적이고, 외국에서는 엘리엇
매니지먼트(삼성물산-제일모직 합병 때 개입한 펀드)가 유명합니다.

이런 펀드가 들어가면 주가가 뛰는 경우가 많습니다 — 그래서 *그들이 관심 가질 만한
종목을 미리 찾는 것* 자체가 투자 전략이 될 수 있습니다.

### 한국에서 지금 왜 중요한가요?

2024~2025년 한국 상법이 개정됐습니다. 핵심은 **"이사는 회사뿐 아니라 주주의 이익도
충실히 챙겨야 한다"** 는 의무가 명문화된 것입니다. 이전에는 행동주의 펀드가 한국에서
잘 통하지 않았지만, 앞으로는 다를 가능성이 높습니다. 이 도구는 그 가능성을 시스템적
으로 추적하기 위해 만들어졌습니다.

## 2. 결과물 미리 보기

도구를 실행하면 두 가지가 만들어집니다.

### (1) 후보 종목 목록 (`report.md`)

수십 개 종목을 신호 강도별로 분류한 표:

| 등급 | 뜻 |
|---|---|
| **강한 후보** | 모든 조건 갖춤. 우선 검토 |
| **유망 후보** | 두 가지 조건이 강함 |
| **시장 회의적** | 큰손 펀드가 이미 들어왔는데 주가가 안 움직임. *역설적으로 가장 매력적* |
| **관심 종목** | 한 가지 조건이 압도적으로 강함 |
| **피해야 할 종목** | 최대주주가 너무 많이 가졌거나, 모회사에 일감 몰아주기로 종속된 자회사 |

### (2) 한 종목 심층 보고서 (`deep_dive_<종목코드>.md`)

다음 항목을 자동으로 작성:

- **매수 근거 한 줄** — 왜 이 종목을 사야 하는가
- 회사 개요와 산업 흐름
- 재무제표 핵심 숫자
- **자회사 가치 합산** — 지주회사라면 보통 부분의 합이 시총보다 30~60% 큽니다
- **동종업계 다른 회사와 비교** — 정말 싼지, 그냥 업종 평균인지
- 최대주주 구조와 외부 큰손들의 움직임
- 외국인 지분 변화
- 임원 보수가 회사 성과와 잘 맞물려 있는지
- 자사주 매입·소각 5년 이력
- 주가를 움직일 만한 다가오는 이벤트
- 추천 매수 가격대와 보유 비중
- **마지막에 반드시 "사람이 직접 확인해야 할 항목" 명시**

심층 보고서 예시는 [`examples/sample_reports/`](examples/sample_reports/) 폴더에
6개 들어 있습니다.

## 4. 빠른 시작

### (1) 설치

(컴퓨터에 Python 3.11 이상이 설치되어 있어야 합니다)

```bash
git clone https://github.com/mgkim1976-spec/activist-scout.git
cd activist-scout
pip install -e .
```

### (2) 자격 증명 4개 준비

이 도구는 외부 데이터 사이트에서 정보를 받아오기 때문에 일부 사이트의 회원 가입과
키 발급이 필요합니다. **모두 무료** 입니다.

```bash
cp .env.example .env
# 그 다음 .env 파일을 열어 4개 칸을 채웁니다
```

| 키 이름 | 어디서 받나 | 용도 |
|---|---|---|
| `KRX_ID`, `KRX_PW` | https://data.krx.co.kr 가입 | 한국 주식 시세·재무 데이터 |
| `DART_API_KEY` | https://opendart.fss.or.kr 가입 후 발급 | 한국 공시 (지분율, 사업보고서) |
| `OPENAI_API_KEY` 또는 `GEMINI_API_KEY` | OpenAI 또는 Google 가입 | 보고서 자동 작성 (인공지능) |

### (3) 실행

```bash
# 후보 종목 목록 만들기 (약 10분 걸림)
python -m activist_scout.pipeline

# 한 종목 심층 보고서 (약 30초)
python -m activist_scout.deep_dive 021820   # 021820은 '세원정공' 종목코드 예시
```

### (4) (선택) 매일 자동 알림

큰손이 5% 이상 지분을 새로 신고하면 알림 받기:

```bash
# crontab 에 추가 — 매일 오전 9시 실행
0 9 * * * cd /path/to/activist-scout && python -m activist_scout.monitor
```

### 후보 종목을 8단계로 거릅니다

| 단계 | 무엇을 보나 | 결과 파일 |
|---|---|---|
| 1. 1차 필터 | 자산 대비 주가가 싸고, 흑자 이어가고, 매출 늘고, 최대주주 지분이 적당한 회사만 | `screening_value_ownership.csv` |
| 2. 기관 매수 흐름 | 최근 90일·20일 동안 기관이 사고 있는지, 평균 매수가가 얼마인지 | `institutional_flow.csv` |
| 3. 거래량 검증 | 펀드가 5% 지분을 모으는 데 며칠이 걸리나 (오래 걸리면 매력 떨어짐) | `liquidity.csv` |
| 4. 공시 정보 보강 | 자사주 거래, 큰손 신고, 거버넌스 사고, 임원 명단, 자회사 등 | `enriched.json` |
| 5. 일감몰아주기 비중 | 인공지능이 사업보고서를 읽어 그룹 내 거래 비중 계산 | `enriched.json` (필드 추가) |
| 6. 회사 성격 분류 | 인공지능이 7가지 카테고리로 분류 (지주회사·승계 진행 중·구조조정 중 등) | `classification.csv` |
| 7. 3축 점수 매기기 | (아래 설명) | `scores.csv` |
| 8. 보고서 출력 | 신호 강한 순으로 정리 | `report.md` |

### 3축 점수란?

각 종목에 대해 세 가지 관점에서 0~100점을 매깁니다.

| 축 | 묻는 질문 | 핵심 신호 |
|---|---|---|
| **타깃 매력도** | 행동주의 펀드가 군침 흘릴 만한가? | 회사가 빚 갚고도 남는 현금이 시총의 30% 이상인가 / 최대주주 지분이 40~50%대인가 (너무 높으면 캠페인 무력) / 자회사 합산 가치가 시총보다 큰가 / 자산 대비 주가가 동종업계 대비 싼가 |
| **이미 매집 흔적** | 누군가 이미 사고 있나? | 최근 90일 중 매수일 비율 / 누적 순매수 / 평균 매수가가 흔들림 없는가 / 분산 매수인가 |
| **법적 지렛대** | 상법 개정으로 새로 생긴 카드가 있나? | 자사주 처분·신탁 해지 / 분할·합병 / 거버넌스 사고 / 일감 몰아주기 / 사외이사 임기 만료 임박 |

### 9가지 등급 분류 (보고서 우선순위 순)

| 등급 | 조건 | 의미 |
|---|---|---|
| **강한 후보** | 3축 모두 60점 이상 | 모든 조건 갖춤 |
| **유망 후보** | 3축 중 2개가 60점 이상 | 두 축 강함 |
| **시장 회의적** | 큰손 신고가 났는데 주가가 시장 평균 대비 5% 이상 못 따라감 | *역설적으로 가장 매력적인 매수 기회* |
| **관심 종목** | 한 축이 70점 이상 압도적 | 단일 축 압도적 |
| **시장 미온적** | 신고 후 시장 평균 대비 -5%~+20% 사이 | 잔여 차익 가능 |
| **이미 반영됨** | 신고 후 시장 평균 대비 +20% 이상 | 늦었음 |
| **신호 신선** | 신고 7일 이내 | 아직 평가 못 함 |
| **관망** | 모든 축 미달 | 현재 신호 없음 |
| **피해야 함** | 최대주주 65% 이상이거나, 모회사 종속 자회사 | 행동주의 무력화 |

## 7. 주요 측정값 설명

이 도구가 보고서에서 자주 쓰는 숫자들의 뜻:

- **잉여 자본 비율**: (현금 + 단기 투자 − 총부채) ÷ 시가총액. **30% 이상이면 행동주의
  핵심 타깃** — "회사가 빚 다 갚고도 시총의 30% 이상이 현금으로 묵혀 있다"는 뜻이라
  주주 환원 압박이 가능
- **자산 합산 할인율**: (시총 − 자회사들 가치 합) ÷ 자회사들 가치 합. **음수면 시총이
  부분 합보다 작다**는 뜻 — 지주회사에서 흔함
- **자회사 가치 신뢰도**: 상장된 자회사 비중이 절반 이상이면 *높음*, 4분의 1 미만이면
  *낮음* (비상장은 장부가만 알 수 있어 부정확)
- **5% 매집 소요일**: 펀드가 평균 거래량 기준으로 5% 지분을 모으는 데 며칠 걸리는지.
  30일 이하면 매력적, 90일 이상이면 사실상 불가능
- **자사주 점수**: 자사주 매입·소각은 +, 신탁 해지·처분은 −. 양수면 주주 우호적
- **신고자 종류**: 행동주의 펀드 / 준행동주의 / 패시브 / 사모펀드 / 전략적 투자자 / 개인
- **피해야 함 면제 규칙**: 최대주주가 60~65%여도 (자산 합산 할인율 -30% 이하) 또는
  (자사주 점수 +1.0 이상) 면제 — 모회사가 깨워야 가치가 나오는 케이스
- **모회사 종속 자회사 식별**: 일감 비중 50% 이상이면서 모회사 지분 50% 이상 →
  자동으로 *피해야 함* 분류

## 8. 솔직히 못 하는 것들

이 도구가 **할 수 없는** 것을 정직하게 명시합니다 (모든 심층 보고서 마지막 섹션에
표시됩니다):

- 공시 문서의 *내용 자체* 를 읽고 해석하는 것 (제목과 첨부 파일명만 분류)
- 외국 행동주의 펀드의 한국 진입 신호 (블룸버그·헤드헌터 네트워크가 필요)
- 경영진 인터뷰
- 노조 동향, 산업 가십
- 동료 매니저·법률 자문 검토
- 부동산 장부가 vs 실제 시세 차이 (별도 데이터 필요)
- *미래 수익률 예측* (어떤 주장도 안 합니다)

**가장 위험한 함정**: 이 도구가 *마치* 의사 결정을 대신해 주는 것처럼 보이는 것.
실제 매매 결정은 **반드시 사람이** 합니다.

## 9. 보완 도구 — 5pct-radar

이 도구를 쓰다가 *"5% 이상 지분 신고 본문 자체를 자동 분석할 수 있겠다"* 는
아이디어가 나와 별도 도구로 분리됐습니다.

**[5pct-radar](https://github.com/mgkim1976-spec/5pct-radar)** — 한국 거래소
*모든* 5% 대량보유 신고 본문을 자동 분석해 시나리오 분류 (행동주의 / 산업 통합
인수합병 / 사모펀드 인수 등) + 12개월 기대값 분포 *초안* 을 만드는 도구.

| | activist-scout (이 도구) | 5pct-radar |
|---|---|---|
| 목적 | 행동주의 펀드 캠페인 *후보 발굴* | 모든 5% 신고 *본문 분석* |
| Universe | 유가증권시장 행동주의 후보 ~48종 | 한국 거래소 전체 5% 신고 (연 수만 건) |
| Cadence | 주간 | 매일 |
